Photo Encryption

Designed and coded a program to implement a new variation of encryption using photos. The program utilizes symmetic encryption to split up the starting image into 3 separate images. To retrieve the message, the pictures can be recombined and using the symmetric key, will decode the message from inside the picture. To insert the message into the photo, the program uses the initialization vector to randomize which pixel the program will start adding the ASCII integer equivalent for each character of the message. So far this only works with PNG format images, as they use lossless compression.
